Agri-Commerce
A minimal e-commerce starter built with Vite, React, and TanStack.

Quick start
-Install:
npm install

- Run dev server:

npm run dev -- --host
- Build:
npm run build

- Preview build:
npm run preview


Notes
- Configure environment variables in .env.
- Supabase settings live in config.toml.
